网站的搭建

安装Node.js

安装git

安装Hexo

`npm install -g hexo-cli`

在指定文件夹中新建所需要的文件

`hexo init <folder>`

安装 hexo-deployer-git

`npm install hexo-deployer-git --save`

在 _config.yml 中修改参数

deploy:
-type: git
repo: <repository url>
branch: [branch]
-type: git
repo: <repository url>
branch: [branch]

配置本地config文件

Host Hostname
HostName git.coding.net
PreferredAuthentications publickey
IdentityFile ~/.ssh/id_rsa
